A high-end, professional currency conversion dashboard designed with a 2026 glassmorphic aesthetic. This terminal provides institutional-grade benchmark rates for global pairings alongside a real-time financial intelligence feed.

## Key Features
* **Institutional Chronometer:** Real-time 24-hour clock and date tracking for time-sensitive operations.
* **Global Coverage:** Benchmark rates for 150+ world currencies.
* **Live Intelligence Wire:** A sidebar featuring high-priority financial news headlines.
* **Glassmorphic UI:** A premium "Midnight & Emerald" aesthetic utilizing high-blur backdrops and neon accents.
* **Optimized Logic:** Instant conversion calculations performed locally for sub-millisecond precision.

## Tech Stack
* **Styling:** Tailwind CSS 
* **Logic:** JavaScript
